# Security Policy

## Authorized Use

Xploit is intended only for systems you own or have explicit permission to assess.

Do not use this tool against third-party systems without authorization. The maintainers are not responsible for misuse.

## Reporting Issues

If you find a vulnerability in Xploit itself, open a private security advisory on GitHub if available, or contact the maintainer directly before public disclosure.

When reporting an issue, include:

- Xploit version
- Python version
- Operating system
- Reproduction steps
- Expected and actual behavior

Do not include secrets, credentials, or private target data in public issues.
